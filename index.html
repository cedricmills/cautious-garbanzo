Regex Creation

Regular Expression, also called regex, regexp, is a sequence of characters for defining a search pattern in string data. It is an independent language specialized in its niche use case and became popular in 1968 for pattern matching in a text editor.

Regex Creation

There are two ways you can create a regex in JavaScript, with a RegExp constructor, or as a literal value. Note, like everything else, regex is also an object in JavaScript. When using a regex literal, you must enclose the pattern between two forward slashed /.

Special Characters

The true power of Regular Expression is in its special characters. You can write regex patterns using a combination of simple and special characters. As an example, the regex /ab*c/ means an 'a' followed by zero or more 'b' (* means zero or more occurrences of the previous item) and then a 'c'. It will match a selection of different strings.
